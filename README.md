# Epigen Project: The effect of hypoxic stress on glioma-associated myeloid cells

This project was part of the course "Bioinformatic Approaches to Regulatory Genomics and Epigenomics" thaught by Dr. Pierre-Luc Germain.  
The goal of this project was to investigate the epigenetic effects of hypoxic stress on glioma-associated myeloid cells.

## Project Information

- **Group members:** Olga Ledeneva, Yara Roth and Julia Shea  
- **Submission Date:** 03 July 2026

## Project Structure
- The `FinalReport.Rmd` (and compiled version `FinalReport.html`) contains our complete workflow including Introduction, fully embedded Analysis code, Results and Discussion for each section and a final Conclusion. 

- We have intially divided our work into separate analysis files (`01_download.Rmd`,`02_differential_anaylsis.Rmd`,`03_GO_enrichment.Rmd`,`04_motif_enrichment.Rmd`). We have retained these if you want to inspect or run specific parts of the analysis in isolation.

The core of our analysis is organized into five distinct parts, with embedded results and discussion for each part:
1. Data download
2. Differential accessibility analysis
3. Functional enrichment analysis
4. Motif enrichment analysis
5. Integration of CUT&RUN data
